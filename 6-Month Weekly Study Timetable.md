6-Month Weekly Study Timetable
--------------------------------

Month 1–2: Python Basics
Week 1–2: Install Python, learn variables, data types, operators
Week 3–4: Practice loops, conditionals, functions
Week 5–6: Work with lists, dictionaries, file handling
Week 7–8: Mini-project: Parse log files from testing domain

Month 3–4: Automation Frameworks
Week 9–10: Learn Pytest basics (test cases, assertions)
Week 11–12: Explore Robot Framework (keywords, test suites)
Week 13–14: Automate simple test cases for a sample web app
Week 15–16: Mini-project: Automate login test and document results

Month 5–6: Cloud Fundamentals
Week 17–18: AWS Cloud Practitioner or Azure Fundamentals basics
Week 19–20: Learn compute (EC2/VMs) and storage (S3/Blob)
Week 21–22: Networking basics in cloud
Week 23–24: Mini-project: Deploy sample app to cloud and test connectivity

Daily Routine (30–45 mins)
15 mins: Watch/Read tutorial
15 mins: Practice coding or cloud labs
15 mins: Log progress in tracker

Outcome After 6 Months
- Comfort with Python scripting
- Ability to automate basic test cases
- Understanding of cloud fundamentals
- Portfolio with at least 3 mini-projects documented on GitHub
